# calendar

Google Calendar Discord bot. Shows today's events, upcoming schedule, and lets you create calendar events from Discord commands. Integrates with Google Calendar API.

## Topics
python, discord, bot, google-calendar, calendar, productivity

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/calendar.git
cd calendar

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 calendar.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
